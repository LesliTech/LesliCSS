Grids

Adaptativo y flexible sistema de grid

Responsive cat utiliza dos tipos diferentes de grids
1.- agrid - grid automatico
2.- mgrid - grid manual

Ejemplo de grid automatico:

<div class="mgrid col-3">
   <div>columna 1</div>
   <div>columna 2</div>
   <div>columna 3</div>
</div>

Resultado final:

columna 1
columna 2
columna 3
