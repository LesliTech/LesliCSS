Grids

Responsive cat utiliza dos tipos diferentes de grids
1.- rgrid - grid automatico
2.- rgrid col-n - grid manual

Ejemplo de grid automatico:

<div class="rgrid col-3">    <div>columna 1</div>    <div>columna 2</div>    <div>columna 3</div> </div>
